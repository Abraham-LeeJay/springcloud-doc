# [SpringBoot与SpringCloud的版本对应详细版](https://my.oschina.net/leejayblog)

大版本对应：

| Spring Boot   | Spring Cloud             |
| ------------- | ------------------------ |
| 1.2.x         | Angel版本                |
| 1.3.x         | Brixton版本              |
| 1.4.x stripes | Camden版本               |
| 1.5.x         | Dalston版本、Edgware版本 |
| 2.0.x         | Finchley版本             |
| 2.1.x         | Greenwich.SR2            |
